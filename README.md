## express-mongoDB-TodoList
Todo List 
## technologies

* node.js,
* express,
* ejs,
* MongoDB, 
* mongoose, 
* bodyParser, 
* loadash, 

## instalation
```node.js
  git clone https://github.com/Manoj8079/TodoApp.git
  run npm instal
  run mongod in the terminal
  run node app.js
  open localhost:3000
  ```
## requirements
  node.js
  MongoDB
